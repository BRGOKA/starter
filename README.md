# Far Away
## far away is a silly serverless web-page project to keep track of the items i need to pack for traveling

## 🛠️ Technologies Used

Nothing crazy just basic ![React](https://img.shields.io/badge/react-%2320232a.svg?style=for-the-badge&logo=react&logoColor=%2361DAFB)

## 📦 Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/BRGOKA/Far-Away.git
   cd Far-Away
   ```

2. **Install dependencies**

   ```bash
   npm i
   ```

3. **Start the development server**

   ```bash
   npm start
   ```
